Without package.json

Build and start hot-releading server (no package.json or node_modules

  1. Create an index.html
  2. Run and serve: npx @11ty/eleventy --serve

It creates _site folder with compiled files

With package.json

  3. Add script section to package.json: 
          "scripts": {
        "build": "npx @11ty/eleventy",
      }
